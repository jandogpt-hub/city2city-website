## ğŸš€ Deployment

This site is deployed via GitHub integration to Hostinger.

### Deployment Steps:
1. Push changes to `main` branch
2. Hostinger auto-deploys via GitHub webhook
3. Files are served from `public_html/`

### Hostinger Configuration:
- **Document Root:** `public_html/`
- **PHP Version:** 8.0+
- **SSL:** Enabled (Let's Encrypt)
